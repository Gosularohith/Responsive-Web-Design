# Responsive Web Design Project

A modern, mobile-first responsive website built using **HTML5**, **CSS3 (Flexbox & Grid)**, and **JavaScript**.  
This project demonstrates best practices in responsive design, accessibility, and performance optimization.

---

## 🚀 Features
- Mobile-first design approach
- Responsive layouts with CSS Grid & Flexbox
- Media queries for multiple screen sizes
- Accessible navigation and semantic HTML
- Optimized images and assets
- Cross-browser compatibility
